# terminus-hapi

Utilizes [terminus](https://github.com/godaddy/terminus) to add graceful shutdown and Kubernetes readiness / liveness checks for applications based on hapi.

## Installation

Install via npm:

```console
$ npm i terminus-hapi --save
```

or yarn:

```console
$ yarn add terminus-hapi
```
